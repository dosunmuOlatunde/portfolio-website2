# WordAnalytics

A modern and responsive word analytics web app that helps users track word count, character count, and social media character limits in real time.

🚀 Features

* Real-time word counting
* Real-time character counting
* Twitter character limit tracker
* Facebook character limit tracker
* Responsive design for desktop and mobile
* Clean and modern UI
* Instant text analysis while typing

🛠️ Built With

* HTML5
* CSS3
* JavaScript

📂 Project Structure
├── index.html      # Main HTML structure
├── tutor.css       # Styling and responsive design
├── tut_01.js       # JavaScript functionality

The app provides:

* A large textarea for entering text
* Live statistics dashboard
* Social media character tracking
* Responsive layout for smaller devices

⚙️ How It Works

The application listens for user input inside the textarea and automatically updates:

* Total number of words
* Total number of characters
* Remaining Twitter characters
* Remaining Facebook characters

If the character limit is exceeded, the counter changes color to red.

📱 Responsive Design

The project is fully responsive and adapts smoothly to:

* Desktop screens
* Tablets
* Mobile devices
